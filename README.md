### setup

1.    Install Dependencies:
      ```bash
      i)    Python3 
      ii)   word2number
      ```
2.    Clone the repo
      ```bash
      git clone https://github.com/asthemus/Bus-reservation-API.git
      pip install word2number
      ```
3.    Run the server
      ```bash
      Python3 main.py
      ```
